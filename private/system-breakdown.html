Cooperative Main Loop

while (running):
    // CAN RX from SocketCAN (non-blocking)
    // Process override commands (0x7E0)

    // 1ms cyclic
    FTSK_RunUserCodeCyclic1ms()           // Timers, DIAG flags
    FTSK_RunUserCodeEngine()               // DATA task
    MEAS_Control()                         // AFE cell voltage/temp queueing

    // 10ms cyclic
    FTSK_RunUserCodeCyclic10ms()           // SYS, BMS, CAN TX

    // 100ms cyclic
    FTSK_RunUserCodeCyclic100ms()          // Balancing, SOC/SOE
    FTSK_RunUserCodeCyclicAlgorithm100ms()

    // Software watchdog: stall >100ms → force ERROR
    usleep(500)  // 500µs per iteration (~2kHz)

Initialization Sequence (6 phases)

  1. Hardware init (all stubbed) + DIAG init
  2. Set ftsk_allQueuesCreated flag (DATA_Initialize passes)
  3. Set os_boot = OS_SCHEDULER_RUNNING
  4. FTSK_InitializeUserCodeEngine (BMS_Init, etc.)
  5. Set os_boot = OS_ENGINE_RUNNING
  6. FTSK_InitializeUserCodePreCyclicTasks → enter main loop
Register Redirection to RAM

All TMS570 hardware registers redirected to 4KB RAM buffers via patches:

// Before (TMS570 hardware)
#define canREG ((canBASE_t *)0xFFF7BC00U)

// After (FOXBMS_POSIX)
extern char posix_canreg[];
#define canREG ((canBASE_t *)posix_canreg)

No segfault on memory-mapped I/O dereference. Register writes/reads are harmless (no side effects).

Closed-Loop Features

  • Reads foxBMS CAN TX (0x220) to detect NORMAL state
  • Only discharges when state 7 (NORMAL) detected
  • SOC tracking: SOC_new = SOC - (I_mA × dt_h / Q_mAh)
  • IR drop: V_pack = OCV - I × R_total

Big-Endian Encoding (Critical Gotcha)

foxBMS uses non-standard big-endian bit numbering (NOT DBC positions). Verified lookup table:

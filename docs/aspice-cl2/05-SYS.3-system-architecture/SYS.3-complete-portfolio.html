Key Finding GAP-03 ASIL D

foxBMS plausibility spread check (PL_CELL_VOLTAGE_SPREAD_TOLERANCE_mV = 300) with WARNING severity (not FATAL) suppresses genuine single-cell overvoltage detection on the ASIL D safety path. A cell with an internal defect exceeding 300 mV above module average is silently invalidated. Accepted as residual risk (<10⁻⁷/hr). Three customer recommendations documented.

CONSTRAINT-001: SPS and SBC share spiREG2. SPS_Ctrl() must execute before SBC_Trigger() in 10ms task. Latent SPI_Lock bypass (GAP-08).
